# Vikram Singh Resume

A simple, static personal resume website for Vikram Singh.

## Overview

This repository contains a single-page HTML resume built with a clean, responsive layout and modern typography.

## Files

- `index.html` — the complete resume page.

## Usage

1. Open `index.html` in a web browser.
2. Or serve it locally using a simple HTTP server:
   - Python 3: `python3 -m http.server 8000`
   - Then visit `http://localhost:8000`

## Notes

- No build step is required.
- The page is self-contained and uses inline CSS.
